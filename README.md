# ChemStruck

Monorepo for chemical drawing application.

## Development

Install dependencies:

```bash
pnpm install
```

Run web app:

```bash
pnpm dev
```

Build all packages:

```bash
pnpm build
```

Run unit tests:

```bash
pnpm test
```

Lint and format:

```bash
pnpm lint
pnpm format
```

Run end-to-end tests:

```bash
pnpm e2e
```
